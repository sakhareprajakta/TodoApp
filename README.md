# Todo App

A simple and responsive Todo application built using React.js and Vite.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Responsive Design

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
